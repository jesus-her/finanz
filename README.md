# Finanz | Crypto/Fintech App

**Run with "yarn start"**

![Cover](https://user-images.githubusercontent.com/88808776/160944619-e4097922-6f42-4b52-8212-391df9d19f7e.png)


## A UI/UX mobile app, made with React Native (Expo CLI). It is an application created to make your investment more comfortable to manage, without long explanations, with a portfolio of modern instruments trying to balance your risk. It is an application with which you can automate your purchases and sales of cryptocurrencies, besides that, it includes the benefits of a Fintech and connects you directly to the website of "cetes directo".
